# Installation

Installing the Smart Irrigation integration consists of a few steps:

1. [Downloading the integration](installation-download.md).
2. Giving the integration a unique name. Setup is a single step — the weather service is no longer chosen during install; you configure it afterwards from the panel.

![](assets/images/installation-1.png)

After the integration has been installed, you will find a new panel named 'Smart Irrigation' in your side bar. [Use it to configure your set up](configuration.md).

Then choose where your weather data comes from:

3. [Configuring the weather service](installation-weatherservice.md) (done from the panel, on the fly).

4. [Options: using manual coordinates](installation-options.md)
5. [Updating to the latest or specific version](installation-updating.md)
6. [Migration from V1 (v0.0.X) to V2 (202X.X.X)](installation-migration.md)
